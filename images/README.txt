SLIIT Logo Instructions
========================

Please place the SLIIT logo file in this folder with the filename:
sliit-logo.png

The logo should be:
- PNG format (or update the file extension in HTML files if using a different format)
- Recommended size: 200x200 pixels or similar square dimensions
- Transparent background preferred
- The CSS will automatically resize it to fit the navbar (40px height) and footer (50px height)

If your logo file has a different name, update the src attribute in all HTML files:
- index.html
- attendance.html
- payroll.html
- agent-profile.html
- application-tracker.html
- roaster.html

Look for: <img src="images/sliit-logo.png" ...>
